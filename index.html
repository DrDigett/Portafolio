Principales proyectos

CriptoExchange

Cripto Exchange es una Single Page Aplication desarrollada en Vue.js. Esta SPA, se conecta a una API para brindar informacion actualizada (cada 18 segundos) del valor de cripto monedas.

Antidoto Edit

Antídoto Edit es la página de una editorial peruana comprometida con la cultura nacional. Este sitio web, desarrollado con HTML, CSS y JavaScript, emplea API's JSON para contenido dinámico. Descubre la pasión por la escritura en este antidoto.

Pelifast

Pelifast: Tu destino cinematográfico. Con JQuery como base, fusionamos HTML y CSS para ofrecerte una experiencia única de navegación. Desde la búsqueda hasta la visualización de películas, cada función está diseñada para brindarte comodidad y entretenimiento.
